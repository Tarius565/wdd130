Overview

Purpose

All in one stop to see the various things that I've made using different softwares on the computer. This site will bring awareness to the tools used and provide a great source to keep track of my creations. In the future this site will have a shop to where individuals can purchase the physical prints or files of the miniatures displayed on this site.

Audience

The audience for my site ranges from 10-100 people. These people range from all different ages. Some learning how to sculpt, others wanting to see other peoples sculpting techniques, and the rest wanting to learn more about how to sculpt.

Content

Home page

Enjoy viewing my artwork as I continually update and include more and more visually appealing pieces of work and also TTRPG miniatures. News will posted here and updates on any future work. The scrolling banner will be updated frequently to house the news.

Images for the Home page

Gallery

Here I will display my artwork in a grid pattern. I will not have any specific links for the pieces however I will include a name or description below each of the images displayed to allow for easy identification of the various angles that the pictures will represent. So each image can be discerned from others I will also include a border around the square area for the images. The description will be similar for each image, "Ice Golem front view 50mm in height".

Images for the Gallery

About Us

This About Us section will include ways to get into contact. The software I use. My educational background. And various other information about myself. "My name is Tanner Levi. I first discovered my love for 3D modeling/sculpting during my bachelors in biomedical engineering. I did some work in CAD design during that program and that's where I learned that I can sculpt and do some amazing things in 3-dimensional space...."
